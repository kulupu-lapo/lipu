# lipu.pona.la

The frontend for [**poki Lapo**](https://github.com/kulupu-lapo/poki/), a library of [Toki Pona](https://tokipona.net/) texts.

## Local development

Install [Node.js + NPM](https://nodejs.org/en/download) on your system. Clone the repo. Run in terminal:

```sh
npm i        # install dependencies
npm run dev  # run the dev server
```
